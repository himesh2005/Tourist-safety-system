# Tourist Safety System

A secure emergency identity and geofencing platform combining:

- Blockchain hash integrity
- Digital signature authentication
- QR-based verification
- Geofencing risk detection
- React frontend
- Node.js backend
- Hardhat smart contract

## Modules
- Identity & Registration
- Blockchain Integrity
- Digital Signature
- Verification Engine
- Geofencing (Nagpur Phase 1)

## Future Scope
- Sepolia deployment
- MongoDB integration
- IPFS support
- Real dataset geofencing
